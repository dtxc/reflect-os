# theroid-os
x86_64 os based on [this os tutorial](https://github.com/cfenollosa/os-tutorial)

## building from source
### requirements
> + gcc
> 
> + nasm
>
> + x86_64-elf-gcc
>
> + i386-elf-gdb (for debugging)
>
> + qemu

### building
`make build && make run`
