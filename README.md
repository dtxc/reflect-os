# theroid-os
x86_64 os based on [this os tutorial](https://github.com/cfenollosa/os-tutorial)

## features
+ keyboard and vga driver
+ dynamic memory

## currently working on
+ vfs driver
+ multi threading and scheduling 
+ user mode

## building from source
### requirements
> + gcc
> 
> + nasm
>
> + x86_64-elf binutils
>
> + qemu

### building
`make build`

`make run` to run the os image
